# v77 국면전환 Whipsaw 보완 계획

## 현재 상태 (2026-04-14)

### v77 국면 규칙: KP_MA200_5d
- KOSPI > 200일 이동평균 = 공격, 미만 = 방어
- 5일 연속 확인 후 전환
- 전환 시 기존 포트폴리오 **전량 청산** → 새 전략 재진입

### 성과
- **2021~2026 (프로덕션 5년)**: Cal=5.23, CAGR=152%, MDD=29% → 훌륭
- **7.8년 (2018~2026)**: Cal=1.34, MDD=72% → 국면전환이 역효과

### 문제: 2018년 Whipsaw
- KOSPI가 MA200 근처에서 장기 횡보
- 5일 확인에도 5회 잘못된 전환 발생
- 전환마다 전량 청산 → 재진입 → 손실 누적 → 연 -20.9%

## 보완 아이디어 (우선순위순)

### 1순위: 버퍼 밴드 (Buffer Band)
```
기존: KOSPI > MA200 → 공격
개선: KOSPI > MA200 × 1.02 → 공격, KOSPI < MA200 × 0.98 → 방어
     가운데 4% 구간: 현재 모드 유지 (전환 안 함)
```
- **원리**: MA200 근처 노이즈 구간을 무시
- **장점**: 가장 직관적, 구현 간단
- **테스트 변수**: 밴드폭 1%/2%/3%/5%
- **주의**: 밴드가 넓으면 진짜 전환도 늦어짐

### 2순위: 쿨다운 (Cooldown)
```
전환 후 최소 N일간 재전환 금지
```
- **원리**: 단기 whipsaw 원천 차단
- **장점**: 2018년처럼 한 달 내 3번 전환하는 것 방지
- **테스트 변수**: 15일/20일/30일/40일
- **주의**: 진짜 급변(코로나 급락 등)에서도 쿨다운 적용됨 → 위험

### 3순위: 버퍼 + 쿨다운 결합
```
버퍼 2% + 쿨다운 20일
```
- 버퍼로 노이즈 제거 + 쿨다운으로 연속 whipsaw 방지
- 단독보다 안전하지만 파라미터 2개 → 과적합 위험 증가

### 4순위: 부분 청산 (Partial Liquidation)
```
전환 시 전량 청산이 아닌, 방어 기준 미달 종목만 청산
```
- **원리**: whipsaw 시 전량 청산의 충격을 줄임
- **장점**: 잘못된 전환이어도 손실이 작음
- **단점**: 구현 복잡, 공격→방어 시 성장주가 방어 기준에 안 맞아서 결국 대부분 청산될 수 있음

### 5순위: 이중 확인 (Dual Confirmation)
```
MA200 5일 + 추가 지표도 동일 방향이어야 전환
예: MA200 공격 + KOSDAQ도 MA200 위 → 공격 전환 확정
```
- **원리**: 단일 지표 노이즈를 복수 지표로 상호 검증
- **후보 지표**: KOSDAQ MA200, RSI(14) > 50, Breadth(대형주 MA126 비율)
- **주의**: v78 서치에서 Breadth에 look-ahead bias 발견됨 → 시점별 대형주로 재계산 필요

### (참고) 기각된 아이디어
- **확인일수 증가 (5→10~15일)**: 2021~2026에서 Cal 5.23→3.52~4.28 하락 확인됨. 좋은 전환도 놓침.
- **더 긴 이동평균 (MA250, MA300)**: 신호가 너무 느려짐. MA200이 이미 장기 지표.
