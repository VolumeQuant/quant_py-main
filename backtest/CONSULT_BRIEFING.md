# 컨설팅 브리핑 — KR 퀀트: 권리락 가격보정 제거 & TTM PER 검증 (2026-06-16)

## 0. 한 줄 요약
(1) **권리락 가격보정(corpaction)이 7.4년 Calmar를 1.74로 반토막** 내고 있었음을 발견 → 제거 권고(메커니즘 규명됨). (2) **PER을 연간→TTM으로 바꾸는 건 측정상 완전 중립**(이득0·비용0, IC~0, Calmar는 재생성 노이즈에 묻혀 분리불가) → 성능근거로는 변경기준 미달.

## 1. 시스템 개요

- 국면 overlay: KOSPI MA20>MA80 5일확인 → boost, 아니면 **defense=현금 100%**(약세장 매수 안 함).
- 밸류 = PER·PBR·PCR·PSR의 z-score 평균. **PER만 pykrx 연간**(사업보고서 기준), PCR·PSR은 TTM(최근4분기). → 형(운영자)이 "PER만 연간인 게 불일치 아니냐, TTM이 낫지 않냐" 제기 = 이번 검증 발단.
- 과열캡(v80.23): `시총 / TTM지배순이익`의 단면 z 중 비싼 쪽만 감점, 가중 0.2. (이미 신선한 TTM이익을 가격반응 밸류로 쓰고 있음)
- 7.4년(2019-01-02~2026-06-12, 1827일) BT. TurboSimulator(entry_fixed sticky). 측정지표 = Calmar.
- ★중요 제약: 3슬롯 집중 전략이라 **BT Calmar 노이즈 ±0.3~0.5**. 작은 차이는 무의미. 팩터 비교는 max-selection 편향(각자 best끼리 비교) 금지 → **고정 운영config + 다른기간(OOS/WF) + IC(노이즈無 예측력)** 로 판정.

## 4. 발견 1 — 권리락 보정이 주범 (corpaction)
**무엇:** OHLCV에서 일수익 <-33% 또는 >+45%(KR 가격제한 ±30% 초과=권리락)를 자동 감지해 이전 주가를 back-adjust(무상증자/분할/병합 불연속 제거). 가격팩터(모멘텀·mom_10·저변동성) 왜곡 제거 목적. **2026-06-12 배포.**
**당시 검증의 한계:** "최근 6주간 매수/이탈 결정 0건 변경 → 무해/안전"만 확인. **7.4년 Calmar는 측정한 적 없음.**

**격리 측정(고정 config, 한 놈씩 OFF):**
| 변형 | Calmar |
|---|---|
| 3개 안전장치 전부 ON | 1.74 |
| **corpaction만 OFF** | **4.31** (+2.57) |
| 일회성페널티만 OFF | 3.62 (+1.88) |
| 밸류트랩필터만 OFF | 3.57 (+1.83) |
| 전부 OFF | 4.14 |
| production(state, 실배포) | 3.83 |
→ corpaction 단독 제거가 전부OFF·production보다 높음 = **단독 주범.** (production이 1.74보다 높은 건 production state 7.4년 대부분이 corpaction 배포 前이라 사실상 corp-OFF 역사이기 때문)

**원인 규명(직접 측정):**
- corp-ON과 corp-OFF의 차이는 **100% 모멘텀 경로**(밸류·성장은 펀더멘털이라 불변).
- 실험1 — 권리락 트리거 종목의 트리거 이후 **실제(보정된) 미래수익**: 하락트리거 fwd20d **−6.06%** / fwd250d **−12.80%** (승률 22~25%) vs 전체평균 +1.0%/+10.4%(승률 44%). → **권리락 직후 종목은 강하게 부진**(KR 무상증자=소형주 펌프 후 시들기). 555건, 광범위.
- 실험2 — 매일 근사 composite top3에서 corp-ON만 매수(보정으로 모멘텀 살아남) vs corp-OFF만 매수: **+2.51% vs +8.66% (차이 −6.15%p/픽 20일)**. → **보정이 부실주를 사게 만든다** 직접 증명.
- 메커니즘: 모멘텀 = 수익률/변동성. 가짜폭락(−46%)이 분자↓ + 분모(변동성)↑ 이중으로 모멘텀 박살, ~12개월 지속 → corp-OFF는 권리락 부실주를 자동회피(우연한 알파). 보정은 이 자연필터를 제거.
- 광범위성: 7.4년 1614건 트리거 / 914종목(전체 3405중). 단일종목 fluke 아님.

**주의(dirty signal):** ±33/45% 임계가 거래정지·유동성 catch-up·데이터오류 등 **가짜 CA도 다수 포착**(상승트리거 644건, 5건+ 트리거 종목 50개 — 정상 CA는 7년 1~2건). → 임계 정밀화 여지 있음.

## 5. 발견 2 — TTM vs annual = 측정상 중립
**corp-OFF baseline, 2×2(가중치 × PER기준), 고정 운영config Calmar:**
| | annual PER | TTM PER |
|---|---|---|
| 분기 1.6배 가중(최근중시) | **4.31** | 3.48 |
| 분기 균등(=진짜 TTM) | 3.17 | **3.54** |
→ **승자가 가중치 따라 뒤집힘.** 밸류 IC는 4경우 전부 ~0(−0.006~−0.009).

**★결정적 confound:** annual·TTM 두 재생성을 종목단위 비교하니, PER과 **무관한** 성장_s(0.55 가중)·모멘텀_s(0.30 가중)도 0.02~0.07 어긋남(재생성 비결정성 + 밸류필터 통과종목 변동 전파). **PER 변경신호(0.15 가중·IC~0)가 이 노이즈보다 작다** → Calmar 차이(±0.4~0.8)를 PER 효과로 분리 불가.
**해석:** boost(강세장)에서 밸류 IC~0 = 거의 죽은 팩터. PER을 신선화(TTM)해도 과열캡(이미 TTM이익 사용)과 중복. → TTM 채택해도 **성능 중립**. "묵은 연간데이터(2026-12 시점에 FY2025=12~24개월 묵음) 회피"라는 철학적 정당성은 있으나, 측정상 개선 아님.

## 6. 잠정 결론 (자문 대상)
1. **corpaction 제거** — 측정+메커니즘 확실, 픽당 −6%p. 변경기준(노이즈 초과·WF·메커니즘) 충족 → 단독 배포 권고.
2. **PER→TTM** — 중립(노이즈). 변경기준 "노이즈 초과" 미달. + "한 번에 하나만" 원칙상 corpaction과 동시 변경 시 효과분리 불가. → 성능근거로는 보류, 철학적 선택이면 별도 단계.
