# rank z-score 유니버스 불안정 문제 (2026-04-09)

## 현상
삼성전자가 하루 만에 composite_rank 11위→58위. 자체 z-score 값 거의 변화 없음.

## 직접 원인
run_daily가 매일 DART 갱신. 4/9에 31건 수집, 29건이 유니버스 내 종목.
이 29종목의 Growth TTM이 새 분기보고서로 변경됨.

같은 날짜(4/8)를 DART 갱신 전/후 FG 실행:
- 갱신 전: 삼성전자 cr=11
- 갱신 후: 삼성전자 cr=62
- 삼성전자 값 변화 없이 51칸 하락

## 구조적 원인
rank z-score는 상대 순위 기반. 300종목을 1~300위로 줄세워 정규분포 변환.

29종목 Growth 원시값 변경 → rank 전체 재배열 → 삼성전자 앞으로 47개 종목 진입.
Growth가 공격 모드 65% 비중이라 composite rank 영향 극대.

## 발생 빈도
- 사업보고서 시즌(3~5월, 8~9월, 11~12월): 매일 20~30건
- 비시즌: 매일 5~10건

## bt에서 안 보인 이유
bt는 전체 DART를 한 번에 로드 → 모든 날짜가 동일 DART 풀.
프로덕션은 매일 DART 증분 → 연속 날짜 간 DART 풀 차이.

## Henry 시스템에서 없는 이유
월 1회 리밸런싱 → DART 변동 월 1회 흡수.
우리는 매일 리밸런싱 → 매일 DART 증분이 rank 흔듬.

## 검증 데이터
- 같은 DART: 삼성전자 cr 62→58 (4칸, 정상)
- DART +31건: cr 11→62 (51칸, 비정상)
