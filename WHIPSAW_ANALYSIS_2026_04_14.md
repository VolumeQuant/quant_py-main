## 4. 핵심 결론

### ⚠️ 사용자 우려가 정확히 맞았다

> "단순히 버퍼두는거만으로 해결이 안될수도 있을거같은데"

**BT 결과**:
- 버퍼 2%는 **모든 구간에서 성과 악화**
- 5.25년 Cal **4.41 → 3.10** (30% 하락)
- 7.8년 Cal **1.37 → 1.10** (20% 하락)
- 2023년 수익 **+139% → +58%** (진짜 전환 시점 지연)
- 2019년 수익 **-24% → -31%** (하락장 초기 방어 지연)

### 맹점: 2018 whipsaw BT 범위 밖

- 7.8년 BT = **2018-07-02 시작** → 2018 whipsaw 4건 (2018-02 ~ 06)은 **BT 범위 밖**
- 모든 시나리오의 2018 연간 수익률 동일 (-1.8%, 하반기만)
- 브리핑의 "2018 whipsaw 연 -20.9% 손실" 주장은 **BT로 직접 검증 불가**

### 2018 H1 BT 확장 시도 실패

- DART 2016 데이터는 936종목 상한 (현재 fs_dart_*.parquet 1892개)
- 2018-01-02 시점 fast_generate_rankings_v2.py로 생성 시 **Growth 팩터 전부 0**
- 원인: 2018-01 시점 TTM YoY 계산용 prev 분기 데이터 부족 → Growth 팩터 사전계산 실패
- → 2018 H1 BT 확장은 **기술적 불가**

## 4.5 이중 지표 (KOSPI + KOSDAQ) 검증

사용자 질문("더 탐색 없어?")에 따라 이중 지표 추가 검증.

**규칙**: KOSPI와 KOSDAQ **둘 다 같은 방향** (둘 다 MA200 위 or 아래)일 때만 전환. 상충 구간은 현재 mode 유지.
**제약**: KOSDAQ 데이터 2020-06~ → 2020-06 이전은 KOSPI만 사용 (fallback).

### 결과 (bt_extended + state 합친 정확 비교)

| 시나리오 | 7.8년 Cal | 5.25년 Cal |
|---------|-----------|-----------|
| **v77 기본 (KOSPI만)** | 1.35 | **4.45** 🏆 |
| 이중 AND fallback C=5 | 1.29 ↓ | 4.22 ↓ |
| **이중 AND fallback C=7** | **1.39** 🏆 (+0.04) | 3.65 ↓ |

### 주의: 초기 "Cal 5.00 발견"은 아티팩트

첫 시도에서 이중 AND C=5가 5.25년 Cal 5.00으로 측정됐으나, 이는 `state/`만 로드해 prev day 데이터가 없어 첫 2일(2021-01-04~05) 수익이 누락된 아티팩트. `bt_extended/` 합친 정확 비교 시 **Cal 4.22** (v77 4.45보다 낮음).

### 결론

이중 지표도 **v77 기본을 명확히 초과하지 못함**. 7.8년 소폭 개선(+0.04) vs 5.25년 명확 하락(-0.80). 트레이드오프.

## 5. 권고

### 1차 결론: 프로덕션 변경 없음 — v77 기본 유지 (오후 추가 탐색 전)

**이유**:
1. 프로덕션은 **최근 5년 성과가 가장 중요** → v77 기본 5.25년 Cal 4.45 최고
2. 어떤 단일 변경도 5.25년 Cal 4.45 넘지 못함
3. 버퍼는 명확히 악화 (사용자 우려 확인)
4. C=7은 7.8년 소폭 개선이나 5.25년 4.45→4.03 트레이드오프
5. 이중 지표는 7.8년 소폭 개선(+0.04) vs 5.25년 명확 하락(-0.80) 트레이드오프
6. 2018 whipsaw 해결은 BT 범위 밖이라 직접 검증 불가
