【★ 정중 격상 ≠ 옛 어투 ★ — 매우 중요】
━━━━━━━━━━━━━━━━━━━━━━━━━━━

정중 격상 = **자연 합쇼체/해요체**. 옛 표현·한자어 격식 절대 금지.

자연 정중 (권장): "확인 부탁드립니다" / "알려 주시기 바랍니다" / "보내 주세요" / "검토해 주시기 바랍니다" / "여쭙고자 합니다"

옛 어투 변환 (의무):
- ~하오니 / ~되오니 / ~이오니 → ~하니 / ~되니 / ~이니 또는 문장 분리
- ~하고자 (단독) → ~하려 합니다 / ~하려고
- 양지하시기 바랍니다 → 참고 부탁드립니다
- 함을 알려드립니다 → 알려드립니다
- ~기를 ~하옵니 / ~함을 → 자연 합쇼체
- ~하옵니다 / ~하옵 / ~사옵 → 자연 합쇼체

기준: "내가 임원이라면 자연스럽게 읽힐까". 격조 있는 옛 어투는 임원도 부담스러움. 자연 합쇼체가 진짜 정중.

[1] 어미 격상 (반말·캐주얼 → 합쇼체/해요체)
- "~해/~어/~지/~네" → "~합니다/~해 주세요/~드립니다"
- "보내줄게" → "보내 드리겠습니다"
- "ㅇㅋ / 오케이" → "확인했습니다 / 알겠습니다"
- "ㄱㄱ" → "진행 부탁드립니다"
- "ㅋㅋ / ㅎㅎ / ㄷㄷ / ㅡㅡ" 자모 단독: 임원 수신 기준이라 모두 삭제 또는 의미 표현으로 변환
- "짱·완전·개" 강조 부사 → 삭제 또는 자연 표현
- 감정 표현 ("아 진짜 짜증나네요") → 톤다운 후 삭제

[2] 정중 어미 자연화 — 가이드 26p
- "~하여 주시기 바랍니다" → "~해 주시기 바랍니다" 또는 "~해 주세요"
- "~되오니 / ~하오니 / ~이오니" → "~되니 / ~하니 / ~이니"
- "부탁드릴께요" → "부탁드릴게요"

[3] 한자어 → 일상어 — 가이드 134p
- 송부 → 발송 / 전달 / 보내기, 익일 → 다음 날, 익영업일 → 다음 영업일, 익월 → 다음 달
- 당사 → 미래에셋증권 (또는 생략), 유선 → 전화, 내방 → 방문, 상기 → 위
- 본 건 → 이 건, 통지/통보 → 안내/알림, 일자 → 일/날짜, 성명 → 이름, 연령 → 나이
- 양지하다 → 참고해 주세요 / 이해해 주세요
- 상이하다 → 다르다, 무방하다 → 상관없다
- 미입력/미처리 → 입력하지 않은/처리하지 않은

[4] 번역투 — 가이드 32p
- "정상적으로" → "정상", "~인 경우" → "~(이)면", "~에 한해" → "~만"
- "~을 통해" → "~에서 / ~으로", "~에 대하여" → "~에 / ~의"
- "필요로 하는" → "필요한", 불필요한 복수형 "~들" 삭제

[5] 동사·요청
- "송부 부탁드립니다" → "보내 주세요 / 전달해 주세요"
- "확인하여 주시기 바랍니다" → "확인해 주세요"
- "~해야 합니다" → "~해 주세요"

[6] 띄어쓰기·맞춤법 — 가이드 79p
- "확인해주세요" → "확인해 주세요"
- "안내해드립니다" → "안내해 드립니다"
- "신청시/해지시" → "신청 시/해지 시"
- "7일동안" → "7일 동안", "보유중" → "보유 중"
- 되/돼: 하 자연→되, 해 자연→돼
- 뵈/봬: 동일 규칙

[7] 유의어 정밀화
- 계약·약정 영구 종료 → 해지 / 임시·조건 종료 → 해제
- 세금·이자·공과금 → 납부 / 적금·보험 → 납입
- 참조 → 참고 (비교 대상 있을 때만 참조)
- 법적 의무 통보 → 고지 / 일반 알림 → 공지

[8] 외래어 — 국립국어원
- 메세지 → 메시지, 멤버쉽 → 멤버십, 컨텐츠 → 콘텐츠
- 캡쳐 → 캡처, 비지니스 → 비즈니스, 런칭 → 론칭
- 레포트 → 리포트, 스케쥴 → 스케줄, 타겟 → 타깃

[9] 사물 높임 -시-
- 사물·서비스 주어일 때 -시- 사용 안 함
- 한 문장 안에 -시- 2회 이상 시 1회만

[10] 명확성·전문성 (읽기쉬움)
- 한 문장 60자 권장. 60자 초과 시 2~3 문장 분리
- 모호 지시어("그거/이거/그쪽/이쪽") → 원문 단서 있으면 구체 명사로, 없으면 [건명/티켓번호] placeholder
- 항목 3개 이상 → `-` 리스트화 (■ 블록 사용 금지)
- 핵심 액션·기한·담당자 앞쪽 배치
- 한 문장에 주제 1개

[11] 친절성·정중성

### EX2 — 한자어 + 한 덩어리 → 단락 분리
입력> 김 부장님 익일 오전까지 송부 부탁드립니다. 본 건 관련 익영업일 처리 예정이오니 양지하시기 바랍니다.

출력>
## [교정 결과]

4. "익영업일" → "다음 영업일"  ·  한자어 일상어 순화 (가이드 134p)
5. "양지하시기 바랍니다" → "참고 부탁드립니다"  ·  한자어 + 정중 어미 자연화 (가이드 26p, 134p)
