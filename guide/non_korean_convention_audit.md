## 2. 작업 1 — 즉시 패치 (정정됨)

3 시스템(CX/직원용/UX) [한국 표기 강제] 박스를 **가이드 backing 명확 항목만**으로 정정:

```
[CX L411-, 직원용 L182-, UX L165- 정정 완료]
- 법조항 인용: 「법령명」 제○조 제○항 또는 제○조의○ (가이드 9-12p, 33-50p, 57-67p verbatim 형식). § 기호 사용 금지 (가이드 PDF 등장 0건)
- 날짜·기간: YYYY.MM.DD 또는 YYYY.MM.DD~MM.DD (가이드 82-117p 명시)
- 통화: 외화 USD 100 형식. 원화 만 원·천만 원·억 원 (가이드 33-50p, 119-128p verbatim)
- 표기 일관성: 한 메시지 안에서 숫자·부호·시간·금액 표기 혼용 금지 (가이드 33-50p, 57-67p "기호·문장부호 일치", "시간·날짜·요일 일치" 원칙)
```

**박스에서 제거된 자체 발명 항목** (백로그로 분리):
- AM/PM 차단·24시간제 강제 → `guide_check_backlog.md` 1.1
- e.g./i.e./etc./cf./vs. 차단 → 1.2 (가이드와 모순)
- em dash 차단 → 1.3
- million/M/K/B 차단 → 1.4
- smart quotes 차단 → 1.5

박스 헤더도 정정: "한국 표기 강제 — 자체 발명 비-한국 표기 차단" → "한국 표기 — 가이드 명시 형식" (자체 발명 강조 제거, 가이드 backing 강조).

**byte 영향**: 박스 ~700 → ~500 byte (-200 byte 절감).
