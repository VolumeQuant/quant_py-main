========== Case 1: 펀드 슬랭 오인 삭제 + 전화번호 환각 ==========

[ASIS - 직원 원본 메시지]

[ASIS 부가 - 사용자 후속 명시]
LMS임. 펀드명이 "그까이꺼 대충하면되요" 임 (비격식 표현이지만 정식 펀드명)

[v5.21 출력 (사용자 보고 사고)]
- 펀드명 "그까이꺼 대충하면되요" 통째 삭제 (사용자 안내 없이)
- ■ 문의 블록 자발 생성하면서 전화번호 "1588-1234" 환각
- 사용자가 "펀드명이다"라고 명시한 후에도 보존 안 됨

[케이스 1 본질 재해석 — 2026-04-27 사용자 통찰]
정식 금융사(미래에셋증권) 펀드명에 "그까이꺼 대충하면되요" 같은 비속어/슬랭이 들어갈 수 없음.
직원이 ① 임시 placeholder를 입력했거나 ② 테스트로 입력했거나 ③ 실수로 잘못된 텍스트가 펀드명 자리에 들어간 것.
어떤 경우든 그대로 출력하면 안 됨. v5.22의 정답 동작:
- 펀드명 자리에 비속어/슬랭 발견 → 그대로 출력 금지
- 출력 상단에 ⚠️ "펀드명 확인 부탁드립니다" 가드 안내
- 본문에서 해당 부분은 [상품명] 플레이스홀더 표시 또는 비움
- 환각 전화번호 0건

[v5.22 출력 - 사용자가 RAG 출력 복붙]
(여기에 채워 넣기)

[검증 체크리스트]
□ 펀드명 "그까이꺼 대충하면되요" 그대로 출력 안 함 (펀드명 가드 효과)
□ 상단에 ⚠️ "펀드명 확인 부탁드립니다" 류 가드 안내 등장
□ 본문 펀드명 자리에 [상품명] 플레이스홀더 또는 비움
