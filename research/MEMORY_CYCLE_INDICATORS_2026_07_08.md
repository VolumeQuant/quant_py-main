# 메모리 사이클 천장 지표 리서치 (2026-07-08, 자율루프 S3 — 표시 계기판 후보, 매매 무관)

결론: "고전적 정점 초입 신호 다수 점등, 확정 트리거 미발동".
- 계약가 모멘텀: Q1 +90~95% → Q2 +58~63% → Q3e +13~18% = 뚜렷한 둔화 (가격 정점을 2~4분기 선행하는 late-cycle 신호)
- 컨센 상향폭 89% = 사상최고 포화 (재평가 여력 소진 신호)
- 총마진 85~86% = 2018 정점(60%) +25%p, 지속불가 수준
- 채널 재고 7~9주 (경고선 8~10주 근접) / 공급사 3~5주 타이트
- 반면 CapEx·한국 반도체 수출(6월 +199.5% YoY)은 아직 가속 — 롤오버 없음
- ★2018년형 패턴: 주가가 실적 정점을 2개 분기 선행 하락 — 삼성 사상최대 실적 발표일에 -9.4% = 재현 우려 (Morgan Stanley "Grim Reaper": 가격상승률 둔화+재고 안정화+EPS상향폭 정점)

표시 계기판 추천 (게이트 아님):
1순위 DRAM 현물가 주간 모멘텀 (TrendForce 무료, 현물→계약→실적→주가 순서상 최선행)
2순위 보유종목 컨센 EPS 방향성 — 기존 fusion_consensus_cache 재사용 (주가↓+컨센↑ 괴리 감지 = 조정vs천장 판별 보조)

출처: Luminix DRAM cycle 분석, TrendForce(3/31·6/10), Morgan Stanley 코멘트(7/7), 관세청 수출, SEMI.
