# 산업지주사 NAV 디스카운트 별도 트랙 — 미래 과제

> 2026-05-14 분석 결과 도출. 현재 v80.6은 SK스퀘어 같은 단순 NAV 알파를 놓침. 본격 작업은 추후 결정.

## 배경

2026-05-14 SK스퀘어 케이스 분석:
- 시총 157조 / NAV 283조 (SK하이닉스 20.07% 지분만으로) = 디스카운트 **44.4%**
- 7.4년간 v80.6 ranking 진입 **0회**
- 원인: 지주사의 매출 vs 영업이익 mismatch (지분법 손익으로 영업이익이 매출의 16배) → G 팩터 outlier → 점수 후속 필터에서 자동 탈락

### 동아엘텍이 1위인 이유 (NAV 알파 + 자회사 폭증의 결합)
- 자회사 1개(선익시스템) → 연결재무제표 = 자회사 매출과 사실상 일치 → mismatch 작음 → v80.6 통과
- 자회사 매출 YoY 4~5배, 영업이익 YoY 15~17배 (8.6세대 OLED) → G 점수 압도적 1위
- 동시에 NAV 디스카운트 58% (지주사 가치 회복 알파까지 동시 작용)

### SK스퀘어가 놓치는 이유
- 자회사 여러 개일 때 연결재무제표 매출과 별도 매출 차이 큼 → DART 매출/영업이익 mismatch
- check_data_mismatch는 DART vs FN 간 비교만, 종목 내부 mismatch는 못 잡음
- G 팩터 outlier로 -1.5σ 또는 (e) capped 트리거 → 자동 탈락
- 즉 **시스템이 명시적으로 거르는 게 아니라 점수 계산이 비정상 outlier로 빠져서 자연 탈락한 부산물**

## 별도 트랙 디자인 (제안)

```
1. universe: 산업지주사 list (수동 30~50개 또는 DART 종속회사 자동 파싱)
2. 매일 NAV 계산:
   - 자회사 지분율 × 자회사 시가총액 → 보유 지분 가치
   - + 기타 자산 - 부채 (DART 별도재무제표)
3. 디스카운트율 = (NAV - 시총) / NAV
4. 점수 = NAV 디스카운트율 (>30%) × 자회사_평균_점수(v80.6 기준)
5. 진입: 디스카운트 30%+ AND 자회사 v80.6 평균 Top 20 이내
6. 퇴출: 디스카운트 10% 미만 (가치 인식 완료)
```

## 예상 알파 vs 한계

**기대:**
- SK스퀘어 같은 단순 NAV 알파 잡음
- 디스카운트 좁혀지는 이벤트 (자회사 IPO, 지분 매각, 합병 등) 알파 큰 폭

**한계:**
- 한국 지주사 디스카운트가 평균 30~60%로 **안정적** → 평균 회귀 알파 약함
- 진짜 알파는 이벤트 트리거 (예측 어려움)
- 자회사 지분율은 연 1회 갱신이라 시점 정확도 떨어짐
- BT 데이터 검증 어려움 (과거 디스카운트율 시계열 부재)
