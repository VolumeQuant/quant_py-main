## 1. 전략 개요

### 전략 A: 마법공식 (Magic Formula)
조엘 그린블라트의 마법공식 기반 **Value + Quality** 전략

**핵심 지표:**
- **이익수익률 (Earnings Yield)** = EBIT / EV
  - EBIT = 세전계속사업이익 (TTM)
  - EV = 시가총액 + 총부채 - 여유자금
  - 여유자금 = 현금 - max(0, 유동부채 - 유동자산 + 현금)

- **투하자본수익률 (ROC)** = EBIT / IC
  - IC (투하자본) = (유동자산 - 유동부채) + (비유동자산 - 감가상각비)

**선정 방식:** 두 지표의 순위 합산 → 낮은 순위(높은 점수) 순으로 선정

---

### 전략 B: 멀티팩터 (Multi-Factor)
**Value + Quality + Momentum** 복합 전략

**팩터 구성:**
| 카테고리 | 팩터 | 산식 | 방향 |
|---------|------|------|-----|
| **Value** | PER | 시가총액 / 당기순이익 | 낮을수록 좋음 |
| | PBR | 시가총액 / 자본 | 낮을수록 좋음 |
| | PCR | 시가총액 / 영업현금흐름 | 낮을수록 좋음 |
| | PSR | 시가총액 / 매출액 | 낮을수록 좋음 |
| **Quality** | ROE | 당기순이익 / 자본 × 100 | 높을수록 좋음 |
| | GPA | 매출총이익 / 자산 × 100 | 높을수록 좋음 |
| | CFO | 영업현금흐름 / 자산 × 100 | 높을수록 좋음 |
| **Momentum** | 12-1개월 | 12개월 수익률 - 1개월 수익률 | 높을수록 좋음 |

**선정 방식:** 각 팩터 Z-Score 계산 → 카테고리별 평균 → 종합 점수 순위
