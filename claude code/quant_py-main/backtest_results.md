### 핵심 발견: 스코어 분포의 양극화

3일 교집합 후보 종목들의 score_100 분포를 분석한 결과:

**중요 패턴**: 3일 교집합 후보의 score_100은 대부분 68~100 구간에 집중됩니다.
이 시스템에서 score_100은 z-score를 100점 척도로 변환한 것으로, 상위 종목들은
자연스럽게 80~100점대에 몰립니다. 결과적으로:

- **score ≥ 72 진입**: 3일 교집합 후보 중 상위 약 절반(~9~11개) 선택
- **score ≥ 80 진입**: 상위 5~6개 선택 → Rank Top5와 거의 동일한 포트폴리오
- **score ≥ 65 이하**: 3일 교집합 후보 거의 전부 진입 → Rank Top20과 동일

이것이 많은 전략이 동일한 수익률을 보이는 이유입니다 — 파라미터가 달라도 실질적으로 같은 종목을 보유하게 됩니다.

### 구조적 분석

**Score vs Rank 핵심 차이:**

- **Score 기반**: 절대적 품질 기준. 시장 약세기에 자연스럽게 전종목 퇴출(현금 보유) 가능.
  → 하락장 방어력 우수. 이 24일 기간에는 현금 보유일이 0일 — 강세장이었기 때문.
  → 실제 하락장에서는 Score 기반이 Rank 기반보다 훨씬 우수할 가능성 높음.
- **Rank 기반**: 상대적 순위 기준. 항상 상위 N개 보유 — 시장 하락기에도 강제 보유.
  → 이 기간(강세장)에는 풀 투자로 유리. 하락장 방어 완전히 불가.
- **Hybrid**: 진입은 엄격(score 또는 좁은 rank), 퇴출은 느슨(score 또는 넓은 rank).
  → 이론적으로 우수하나 24일 데이터로는 검증 불가.

**결론적으로, Rank 기반이 이 기간 더 좋아 보이는 것은 24일이 전부 상승장이었기 때문입니다.**
하락장이 포함된 더 긴 기간에서는 Score 기반(현금 보유 가능)의 장점이 드러날 것입니다.
