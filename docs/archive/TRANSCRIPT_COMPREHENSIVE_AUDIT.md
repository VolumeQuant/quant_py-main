## 시스템적 차이 (재수집 효과 없음)

### q 영업CF 누적 vs 분기단독
- DART의 q 영업CF = 누적값 (Q1=Q1, H1=Q1+Q2, Q3=Q1+Q2+Q3)
- FN의 q 영업CF = 분기 단독
- 두산 등 44종목 부호 다름 = **시스템적 차이, 매핑 사고 X**
- 매출/영업이익/당기순이익 q는 분기 단독 일치 (정상)
- 영향: ranking 사용 시 z-score 정규화로 미미. 추가 작업 불필요.

### q 영업이익 부호 부분 다름 (카프로 등)
- DART q 영업이익 분기 변환 시 음수 부호 처리 오류 가능
