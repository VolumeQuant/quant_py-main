## 기각 아카이브 (재시도 금지)

측정하고 버린 아이디어들. 같은 검증을 두 번 하지 않기 위한 기록이며, "무엇이 안 되는지"는
"무엇이 되는지"만큼 값집니다.

- **유동성 필터 강화** — 얇은 종목 코호트가 오히려 수익 우위. 막으면 승자(브이엠 +28%,
  에스에이엠티 +140%) 학살
- **가격/거래량 함정 필터 20+종** (이격도·거래량 급증·Amihud·상한가 스파이크·자산 증가 등)
  — 깨끗한 음의 신호 0. 함정과 승자는 진입 시점 가격 시그니처가 동일
- **조기위험탐지 8각도** (브레드스 하드게이트·드로다운 브레이커·빠른 MA·변동성 4종) —
  baseline을 이긴 것 0. 시스템 내 조기경보는 휩쏘 비용 > 보호 이득
- **밸류 sleeve / 기대성장 점수 반영 / gap 자격컷** — 기대성장 알파는 비중 조절로만 유효,
  종목 선택에 넣는 순간 전부 실패 (컨센 커버리지 편향)
- **고모멘텀 −1.5σ 면제 / MA60 완화 / 시총 하한 상향** — 유니버스 레버로는 MDD를 못
  줄임. MDD는 분산·게이트·현금이 결정
- **타이트 손절(−8/−10%) / 트레일링 스탑 / 분할매수 / 진입 확인 강화** — 전부 승자를
  같이 깎아서 열위
- **TTM 밸류 전환** — annual과 동등(노이즈 내). 3차례 끝장 검증으로 종결
- **휩쏘 축소 시도 일체** — 진입의 30%가 3일 내 이탈하지만, 휩쏘와 승자는 진입 시그니처를
  공유해 분리 불가. 휩쏘는 본전(평균 −0.6%)이라 막아도 무익
