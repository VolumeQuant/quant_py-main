## 기각 아카이브 (재시도 금지)

측정 후 버린 아이디어들 — 같은 것을 다시 검증하지 않기 위한 기록:

- **유동성 필터 전부** (거래대금 바닥 상향/회전율): 얇은 종목이 오히려 수익 우위, 막으면 승자 학살
- **가격/거래량 함정 필터 20+종** (이격도·거래량급증·Amihud·상한가 스파이크 등): 깨끗한 음의 신호 0 — 필터링 프론티어는 회계 착시까지
- **밸류 sleeve / 점수 반영 / gap 자격컷**: 기대성장은 비중 조절로만 유효, 종목 선택 반영은 전부 실패
- **조기위험탐지 8각도** (브레드스 하드게이트·드로다운 브레이커·빠른 MA·변동성 계열): baseline을 이기는 것 0 — 시스템 내 조기경보는 휩쏘 비용 > 보호 이득
- **고모멘텀 -1.5σ 면제 / MA60 완화 / 시총 상향**: 유니버스 레버로는 MDD를 못 줄임 (분산 상실)
- **타이트 SL(−8/−10%) / 트레일링 스탑 / 분할매수 / 진입 확인 강화**: 승자를 같이 깎아서 전부 열위
- **TTM 밸류 전환**: annual과 동등 (노이즈 내) — 바꿀 이유 없음
