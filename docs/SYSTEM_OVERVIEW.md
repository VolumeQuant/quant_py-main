## 1. 설계 철학과 검증 원칙

**엣지의 정체**: 성장 가속 리더에 공격적으로 참여하는 것. 수익의 75%가 상위 5% 거래에 집중되고(승률 53%, 손익비 3.8), 승자 몇 개를 놓치면 평범해진다. 그래서 이 시스템의 필터·페널티는 전부 "승자를 안 죽이는가(Leave-One-Winner-Out)"를 통과해야 배포된다.

**배포 철칙** (실측 사고에서 나온 규칙, [사고 기록](VERSION_HISTORY.md#주요-사고와-교훈) 참조):

- 모든 신규 룰: **7.4년 풀BT + Walk-Forward 블록(강세/약세/최근) + LOWO + 인접 파라미터 CV** 통과 필수. 단기(수주) 검증만으로 배포 금지.
- BT 노이즈는 Calmar ±0.10 — 이걸 못 넘는 개선은 채택하지 않는다.
- 페널티 효과 측정은 편법 un-apply 금지, 반드시 킬스위치 풀재생성으로.
- 팩터 비교는 best-vs-best 금지(선택 편향) — 고정 config + 여러 기간.
- "고정 사실"(과거 이벤트)은 append-only 누적, 매일 전체 재계산 금지(입력 오염에 취약).
- 모든 배포에 킬스위치(env) + git revert 롤백 경로 명시.

**기각의 기록도 자산**: 유동성 필터, 가격/거래량 함정 필터 20+종, 밸류 sleeve, 고모멘텀 필터 면제, 조기위험탐지 8각도 등 — 전부 측정 후 기각. 같은 아이디어를 두 번 검증하지 않기 위해 기각 근거를 남긴다.

## 2. 국면 게이트

- **신호**: KOSPI 종합지수 MA20 > MA80 = 공격(boost), 미만 = 방어(defense). **5일 연속 확인** 후 전환.
- **전환 시 전량 청산**. 방어 모드 = 현금 100% (신규 매수 없음, 보유 종목만 룰대로 청산).
- 근거: 단기/장기MA×확인일 75조합 재탐색에서 20/80/5가 약세장(2022-23) WF 최소 Calmar 1위. 합산이 더 높은 후보들은 약세장 붕괴(강세장 과적합).
- 코스피200·코스닥150·동일가중·ex-메가 등 대체 게이트는 OOS 결판으로 전부 기각 — 코스피 종합 유지 확정.
- yfinance 장기 데이터로 2000년 이후 6개 약세장에서 매수보유 대비 MDD −7~32%p 방어 확인 (2008년 −32%p가 압권, 빠른 V자(2011)만 약함).
- 전환 진행 중이면 Signal 푸터에 카운트다운 표시 (매매룰 불변, 1일째부터 표시만).

## 4. 팩터 모델

**공격 모드 점수**:

```
score = V×0.15 + G×0.55 + M×0.30
        + mom_10_z×0.05 + vol_low_z×0.06     # 가격 자연 반영 (v80.20)
        + min(과열_z, 0)×0.2                  # 과열 캡 (v80.23)
        − 함정 페널티들                        # 5절
```

- **V (밸류)**: pykrx 연간 PER/PBR 기반. TTM 전환은 끝장검증 결과 동등 → annual 유지.
- **G (성장) — 가중 0.55로 핵심 팩터**: rev_z(매출 YoY) 0.4 + oca_z(영업현금흐름 변화) 0.4 + gp_growth_z(매출총이익 성장) 0.2. 전부 DART PIT(rcept_dt ≤ 기준일).
- **M (모멘텀)**: 12개월. (6m-1m 등 변형은 재검증에서 열위.)
- **mom_10 / vol_low**: 10일 수익률 z + 저변동성 z 가산 — 분기 데이터(90일 불변)로 순위가 박제되는 문제를 가격이 매일 풀어줌.
- **과열 캡**: 자기 PER(시총/TTM 지배순이익)의 단면 z 중 **비싼 쪽만 감점**(싼 것 무보상). 주가가 실적 대비 폭등하면 감점↑ → 추격매수 회피. cheap 보상 변형은 전부 기각(모멘텀 희석).
- **Q(퀄리티) = 0**: 12시나리오 재최적화에서 일관되게 0이 최적.

## 5. 함정 필터 스택

과거 실계좌 손실 사례(선익시스템·삼지전자·디바이스·대덕·에스에이엠티)에서 역설계된, **구조적 회계 착시**만 잡는 3+2중 방어. 가격/거래량 기반은 전수 검증에서 전부 기각됐다.

| 필터 | 조건 → 조치 | 잡는 유형 (실사례) |
|---|---|---|
| **lumpiness** (v80.29) | min/max(최근 4분기 매출) < 0.25 → 성장점수 ×0.3 | 대형 수주가 특정 분기만 터지는 프로젝트 회사 (디바이스: 53→262→86→392억) |
| **accruals** (v80.25) | B=(순이익TTM−영업CF)/자산>25 AND C=max분기영익/TTM>0.7 → 성장 ×0.24 | 일회성 이익으로 TTM 뻥튀기 (삼지전자·에스에이엠티) |
| **최근 CA 페널티** (2026-06-18) | 126일 내 하락 CA(무상증자/분할/유상증자) → 점수 −0.3 | 권리락 직후 부실주 (fwd20d −6%/fwd250d −13% 코호트) |
| **얇은 밸류트랩** (v80.27) | PER>40 & PBR<1.5 & 거래대금<50억 → 제외 | 이익 없는데 장부만 싸 보이는 지주/트랩 (대덕) |
| **QoQ 페널티** (v80.12) | 강한 강세장에서 영업이익 QoQ<+20% → G×0.7 | 과열장 실적 미달 |
| 펌프 게이트 (v80.31) | 이격도20>1.4 AND 저성장 → 매수 차단 | 실적 없는 가격 펌프 |

- 계절성 페널티는 lumpiness와 완전 중복으로 판명 → **제거**(2026-06-25).
- 가격 데이터는 **KRX 공식 수정주가**, 자작 corp action 보정은 잘못된 보정(진짜 폭락을 무상증자로 오인)으로 판명되어 OFF.
- 고PBR 모멘텀 펌프(제주반도체·타이거일렉 유형)는 **의도적으로 필터 안 함** — 승자와 구별 불가라, 진입≤3 + 3슬롯 분산 + 이탈>5 구조로 관리.

## 6. 순위 체계

- **cr (composite_rank)**: 당일 단독 순위. 판단에 절대 안 씀 — wr 입력값 + 궤적 표시용.
- **wr (weighted_rank)**: `cr_T0×0.4 + cr_T1×0.35 + cr_T2×0.25`. **모든 매매 판단의 유일한 기준.**
- **✅ 검증**: cr Top30을 3일 연속 유지해야 매수 후보 (⏳ 2일 / 🆕 1일).
- 순위 기반인 이유: KR 시장 ~1,900종목에서 점수 표준편차 노이즈가 커서 순위가 명확히 우월 (점수 기반 BT Calmar 2.62 < 순위 3.39).
- 표시 점수는 고정 앵커(ws/1.9×100) — 1등 무조건 100점 박제 문제 해결, 매매 무관.

## 7. 매매 룰

| 룰 | 값 | 근거 |
|---|---|---|
| 진입 | ✅ 종목 중 wr ≤ 3 | E2/E4 열위 |
| 이탈 | wr > 5 | 브레드스 도입 후 X6→X5 재최적 (브레드스 규율 준수 전제) |
| 슬롯 | 3 (동일가중 진입) | S4/S5 전체 열위. 3슬롯 분산 자체가 포지션 리스크 컨트롤 |
| 손절 | **−15%** (v80.35) | 테일 보험: 비용 0으로 최악 단일거래 −34.9%→−19.9% 절단. 타이트 SL(−8/−10%)은 여전히 손해 |
| 트레일링 스탑 | 없음 | 회전만 늘리고 성과 열위 (v80.21 제거) |
| 재진입 쿨다운 | 이탈(순위/SL) 후 **10거래일** | 되사기 코호트 승률 29%/평균 −2.05%. K 재스윕: 짧으면(3~5일) 손실, 10~20일 고원 |
| 빈 슬롯 | **승격 금지** (4위로 안 채움) | 승격 허용 시 Calmar 4.77→3.93 악화 |
| 국면 전환 | 전량 청산, 쿨다운 미적용 | 재진입은 한 번에 전량 (분할매수는 알파 아님 — BT 동급) |

SL 이탈 후 반등은 통계적으로 **데드캣**: +3~5일 중앙 +5~7%(승률 86%) 반짝 후 +20일 중앙 −7.5%(승률 29%) — 쿨다운이 이 구간의 추격 재매수를 차단한다.

## 8. 섹터 브레드스

- **신호**: 23개 KRX 섹터 동일가중 지수 중 200DMA 위 비율 < 35%가 3거래일 지속 → **노출 50% 축소 권고**. 35% 이상 3일(보수화: 복귀 5일) → 복귀.
- **역할**: 게이트(MA20/80)가 못 잡는 "느린 약세" 정밀 보호. 조기경보가 아님 — KR 게이트는 이미 빠른 편이라, 더 일찍 당기는 변형(ROC 등)은 휩쏘 비용이 보호 이득을 초과해 전부 기각.
- **binary(전량 방어)가 아니라 50% 스케일**인 이유: 브레드스는 저정밀 신호(헛방어 다수) — 확실한 게이트는 전량, 애매한 브레드스는 절반.
- 매매 신호(3종목)는 불변 — 사이징 권고이며, 시스템 수익률 리플레이에는 기계 반영됨(BT==production 정합).

## 9. 데이터 파이프라인

```
평일 16:00 (Task Scheduler) run_daily.py
  → git pull (다른 PC 변경 자동 반영)
  → 거래일 판정 (pykrx + yfinance 교차확인 — 빈 응답 오판 방어)
  → data_refresher: 시총/펀더멘털/OHLCV 증분/섹터/KOSPI (+staleness 경보)
  → FG 스코어링 (boost + defense 병렬 subprocess)
  → weighted_rank 후처리
  → 확신가중 fusion 갱신 (보유 종목 컨센 수집)
  → 텔레그램 발송 + state/ git push
```

- **재무**: DART 주도(list API 증분 — 그날 공시 종목만) + FnGuide 보충(누락 계정 자동 합침) + 항목별 mismatch 자동 정정. 분기마감일 폭주 대비 document API 폴백(반기 포함, 2026-07 사전 검증).
- **가격**: KRX 수정주가 (`all_ohlcv_adj_*`), 실패 시 raw 폴백. PER/PBR/ROE는 pykrx(KRX 공식).
- **PIT 보장**: 성장 팩터는 실제 공시일(rcept_dt) 기준 — 미래 데이터 누수 차단.
- **안전망**: 랭킹 종목 수 < 150이면 채널 발송 차단 + 재시도 (데이터 사고 감지선). 무소음 실패 경로들은 개인봇 경보로 명시화 (SPOF 감사 전 항목 완료).
- **외부 API 원칙**: pykrx/DART 병렬 호출 절대 금지, 1초 sleep 순차.

## 11. 메타 자산배분

시스템 밖, 사용자 판단 영역 (시스템 코드와 무관):

- **현금 20% / 시스템 80%** (시스템 내 KR 60 : US 40). 분기 첫 거래일 정기 + ±5%p 밴드 리밸런싱, 2%p 무거래 밴드.
- 현금 버퍼는 수익 장치가 아니라 **리스크 다이얼** — Calmar 최적 현금은 기간 의존(폭등기 0%, 정상+약세기 30~40%)이라 성장 목표 감안 20% 선택.
- US 40%의 진짜 이유는 분산이 아니라 **달러 쿠션** — KR/US 주식은 위기에 동반 하락(corr 0.7~0.9)하지만, 원화 약세가 달러 자산의 원화 손실을 축소 (2008: KOSPI −50% vs 나스닥 원화환산 −28%).
- 섹터 쏠림(top3 전부 반도체)이 나도 시스템에 비반도체를 억지로 끼우지 않는다 — 쏠림 방어는 시스템 밖 3층(3슬롯/브레드스/현금·US)이 담당.

## 12. 알려진 한계

- **라이브 검증 5개월** (2026-02-23~): 팔로워 원장 +26% vs KOSPI +25% — 아직 벤치마크 수준. BT의 초과 성과가 라이브에서 재현되는지는 시간이 답한다. 약세장(defense 발동) 라이브 통과 경험 0회.
